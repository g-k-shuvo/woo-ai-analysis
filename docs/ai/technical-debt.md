## Conventions
When adding debt, use this format:
```
### [DEBT-001] Short description
- **Severity**: Low | Medium | High | Critical
- **Area**: Backend | Plugin | AI | Infrastructure
- **Added**: YYYY-MM-DD
- **Context**: Why the shortcut was taken
- **Impact**: What problems it causes or could cause
- **Fix**: How to resolve it properly
- **Blocked by**: Any dependencies preventing the fix
```
